# 🚀 Smart Snake Game v6.0.0
## Codename: portalserpent — Wrapping Border Physics Game Modes

**Release Date**: July 16, 2026

---

### Highlights
- Introduced toroidal coordinate wrapping (Portal Mode) alongside standard solid border death physics.
- Added a Border Physics selection dropdown in the SPEED & SETTINGS sidebar.
- Updated pathfinding algorithms (A* & BFS safety) to solve routes through portal boundaries.
- Adapted Q-learning exploration sensors to detect dangers across wrapping borders.

### Added
- Toroidal coordinate wrapping translation logic inside `GameController.java`'s tick movement.
- Border Mode dropdown selection combo box inside `Project.java`'s options sidebar.
- Toroidal distance heuristics and neighbors expansion checks in `Pathfinder.java`.
- Toroidal wrapping sensor detections in `QLearningAgent.java`.
- Overloaded backward-compatible methods in `Pathfinder.java` and `QLearningAgent.java` to prevent compilation errors in legacy monolith files.

### Changed
- Refactored compiler and packaging outputs to compile directly to the root `out/` folder, removing IntelliJ's nested `out/production/...` layouts.
- Deleted empty, unused boilerplate folders to keep workspace clean.

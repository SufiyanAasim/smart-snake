# Smart Snake — v6.0.0

> *Codename: Portalserpent*

---

## 📖 Overview

**Portalserpent** adds toroidal border physics to Smart Snake — the snake can now teleport through one wall and emerge from the opposite side, just like the portals in classic arcade games. Both the A\* pathfinder and Q-learning agent are updated to solve routes that cross portal boundaries.

Building on the polished UI of **Glowserpent**, this release focuses on game physics — adding an entirely new movement mode, making all AI layers portal-aware, and cleaning up the build output structure.

---

## 🎯 Objectives

* Introduce toroidal coordinate wrapping (portal teleportation) as a selectable border physics mode.
* Make A\* pathfinding and BFS safety fully portal-aware with correct toroidal distance heuristics.
* Update Q-learning danger sensors to detect hazards across wrapping borders.
* Clean up compiler output structure, removing IntelliJ's nested `out/production/…` layout.

---

## ✨ Added

### 🌀 Toroidal Portal Border Mode
* New **Wrap Borders** option alongside existing **Solid Borders** in the SPEED & SETTINGS sidebar dropdown.
* When active, snake movement wrapping logic in `GameController.java` translates coordinates modulo the grid dimensions — the snake exits one edge and enters the opposite.
* Visual feedback: the snake appears to teleport cleanly with no death penalty.

### 🧭 Portal-Aware Pathfinder
* `Pathfinder.java` — toroidal Manhattan distance heuristic: selects the minimum of direct or wrap-around distances on each axis.
* Neighbour expansion in A\* and BFS now includes cells accessible through all four wrap-around edges, not just the four cardinal neighbours.

### 🤖 Portal-Aware Q-Learning Agent
* `QLearningAgent.java` — three danger sensors (straight, left, right) updated to check for walls and body across wrapping boundaries.
* Food direction bits updated to encode the shortest toroidal direction (left-or-right through the wrap may be shorter than direct).

### ↩️ Backward-Compatible Overloads
* Overloaded method signatures added to `Pathfinder.java` and `QLearningAgent.java` so existing callers without the `wrapBorders` flag compile without errors.

---

## 🔄 Changed

* **Build output path** — compiler and JAR packaging now output directly to the root `out/` folder. The old IntelliJ nested layout (`out/production/Project/`) is removed.
* **Workspace cleanup** — empty, unused boilerplate folders deleted from the repository root.

## 🏁 Release Summary

Portalserpent adds a genuinely new dimension to Smart Snake gameplay. Portal mode changes how the snake — and every AI layer — thinks about space: there are no walls, just wrapping coordinates. The pathfinder and Q-agent handle it transparently, so players can switch border physics mid-session and watch the AI adapt instantly.

# Smart Snake — v4.0.0

> *Codename: Vaultserpent*

---

## 📖 Overview

**Vaultserpent** adds relational persistence to Smart Snake. Every game session is now logged to a local SQLite database, and a full interactive leaderboard dialog lets players search, delete, and export their match history.

Building on the clean MVC foundation of **Synthserpent**, this release focuses on data — making score history permanent, queryable, and exportable.

---

## 🎯 Objectives

* Persist every session's score, mode, steps, and timestamp to a local SQLite database.
* Provide an interactive leaderboard dialog with search, deletion, and CSV export.
* Prevent SQL injection via parameterized statement execution.
* Bundle all database dependencies directly in the repository — no user installation needed.

---

## ✨ Added

### 🗄️ Relational Score Database
* `DatabaseManager.java` manages all SQLite interactions via JDBC.
* Relational schema: `id`, `player_name`, `score`, `high_score`, `controller_mode`, `moves_count`, `date_played`.
* Auto-creates the database and table schema on first boot — no manual setup required.
* Parameterized SQL statements prevent injection attacks.

### 🏆 Interactive Leaderboard Dialog
* `LeaderboardDialog.java` — full-featured score browser launched from the sidebar.
* Styled `JTable` with neon-themed custom column headers.
* **Name search bar** — filters rows by partial player name match.
* **Delete Row** — removes a selected score entry.
* **Clear All** — wipes the entire score history after confirmation.
* **Export to CSV** — saves the complete history as a comma-separated file.

### 🔘 Sidebar Scores Button
* Added a **Scores** action button to the sidebar button panel.
* Automatically pauses an active game before opening the dialog.

### 📁 Database Path Configuration
* Default path: `data/scores.db`.
* Overridable via `DB_PATH=` in the `.env` configuration file.
* Parent directory is created automatically if absent.

---

## 🔄 Changed

* Game-over flow now prompts for a player name before saving the score.
* `NameInputDialog.java` introduced for styled name entry on game end.

## 🏁 Release Summary

Vaultserpent makes Smart Snake's score history permanent. The SQLite integration is self-contained — the JDBC driver is pre-bundled and the schema initializes automatically — so players never need to install anything extra. The leaderboard dialog gives the data a rich, searchable UI, completing the persistence arc.

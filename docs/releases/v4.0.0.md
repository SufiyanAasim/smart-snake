# 🚀 Smart Snake Game v4.0.0
## Codename: dbserpent — Scoreboard Relational Persistence

**Release Date**: July 16, 2026

---

### Highlights
- Persistent SQLite database mapping player scores.
- Interactive Leaderboard Dialog with lookup filtering.
- Data utility features including deletion, wipes, and CSV exports.

### Added
- Local SQLite database relational storage manager `DatabaseManager.java`.
- Parameterized statement query execution preventing SQL Injection.
- Interactive Leaderboard dialog scoreboard JTable.
- Sidebar Dashboard button to open the match history ledger.
- CSV scores data exporter.

### Database
- Relational schema containing matching logs: Date, Name, Score, High Score, Control Mode, Moves Count.
- Auto-migration support on boot.

### UI/UX
- Search bar to search scores by player name.
- Custom Swing table headers and neon highlights.

### Infrastructure
- Pre-packaged `lib/sqlite-jdbc.jar` integrated in local power shell compile scripts.

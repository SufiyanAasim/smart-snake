# 🚀 Smart Snake v5.0.0
## Codename: Glowserpent — UI/UX & Resizing Refinements

**Release Date**: July 16, 2026

---

### Highlights
- Fixed the score reset bug to ensure fresh games start at `0` instead of `1`.
- Introduced Render-to-Texture (BufferedImage) display scaling supporting arbitrary window resizes.
- Added support for borderless Fullscreen layout toggle (F11 and sidebar action button).
- Implemented a custom dark-themed player name input modal on Game Over.
- Swapped developer credits to align with lead roles.

### Added
- Custom dark neon theme dialog `NameInputDialog.java` to prompt for player name.
- Sidebar action button `Fullscreen` to toggle screen modes.
- Keyboard `F11` key listener hook to switch undecorated maximized frame states.
- Live slider labels showing active speed multiplier and epsilon exploration floats.
- Window icon properties and taskbar logo support.

### Changed
- Decoupled `GameModel.java` score mapping from raw snake node sizes: `Math.max(0, snake.size() - 1)`.
- Swapped credits in `CreditsDialog.java` and added the active version label to the dialog header.
- Scaled `GameView.java` paint output onto an internal `800x600` canvas before rendering.

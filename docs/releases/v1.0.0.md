# Smart Snake — v1.0.0

> *Codename: Retroserpent*

---

## 📖 Overview

**Retroserpent** is the genesis release of Smart Snake — a classic keyboard-controlled snake game built from scratch in Java 21 using Swing. It establishes the core game loop, rendering pipeline, scoring system, and build infrastructure that every subsequent release builds on.

---

## 🎯 Objectives

* Deliver a fully playable, classic snake experience in pure Java Swing.
* Establish a reliable `javax.swing.Timer`-based game loop.
* Implement directional input buffering to prevent instant self-collision.
* Set up a PowerShell-based build and run pipeline.

---

## ✨ Added

### 🎮 Core Gameplay
* `javax.swing.Timer` game loop running at 20 ticks per second.
* Directional input buffer preventing immediate reversal and self-collision.
* Snake grows by one node on food consumption.
* Randomized food placement that avoids occupied cells.

### 🎨 Rendering
* Custom Swing `paintComponent` rendering layer.
* Green-to-dark-green trailing gradient tail animation.
* Centered AWT text for menu prompts and Game Over notification.

### 🏆 Scoring
* Session-based high score tracker resetting on application close.
* Score increments on each food eaten.

### 🛠️ Infrastructure
* PowerShell compiler script (`build_and_run.ps1`) for runnable JAR compilation.
* NetBeans and IntelliJ IDEA project configuration files for Java 21.

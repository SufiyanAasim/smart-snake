# 🚀 Smart Snake v1.0.0
## Codename: Retroserpent — Genesis Baseline

**Release Date**: July 16, 2026

---

### Highlights
- Establish the baseline classic playable game.
- Dynamic color trail rendering and input buffer logic.
- Target platform: Java 21 support.

### Added
- Standard Swing game loop utilizing `javax.swing.Timer` at 20 ticks per second.
- Custom rendering layers displaying a trailing green gradient tail.
- Session scoring system displaying the session's highest score.
- Input buffers preventing immediate self-collision.
- Powershell compiler script (`build_and_run.ps1`) for runnable JAR compilation.

### Infrastructure
- NetBeans and IntelliJ configuration settings for Java 21 compatibility.

### UI/UX
- AWT-centered text printing for menus and game over notifications.

### Compatibility
- Requires JDK 21+ on Windows, macOS, or Linux.

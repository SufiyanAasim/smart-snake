# Smart Snake — v7.0.0

> *Codename: Battleserpent*

---

## 📖 Overview

**Battleserpent** is the current release of Smart Snake — a major gameplay milestone that adds a rival AI competitor snake, a programmatic arcade synth audio engine, power-up target variations, a real-time visual theme customizer, and a mouse-driven map obstacle painter.

Building on the portal physics of **Portalserpent**, this release transforms Smart Snake from a single-player AI showcase into a spatial combat arena where the player competes against an autonomous rival that runs its own A\* + BFS engine.

---

## 🎯 Objectives

* Spawn an autonomous competitor snake that hunts food and actively blocks the player.
* Add a zero-dependency programmatic MIDI synth engine for arcade sound effects.
* Introduce power-up targets (Golden Apple, Shield Orb) that change the strategic calculus.
* Provide a real-time visual theme switcher with three distinct rendering stylesheets.
* Allow players to draw custom wall layouts directly on the grid without leaving the app.
* Tighten the UI: dynamic Pause/Resume toggle, auto-pause on dialog open, clean Exit button.

---

## ✨ Added

### ⚔️ Rival Competitor AI Snake
* **Battle Arena** — an autonomous competitor snake spawns on the grid when **Spawn Rival AI Competitor** is enabled from the sidebar.
* **Chase Physics** — the rival runs A\* pathfinding with BFS safety to hunt food, grow, and block the player's path.
* **Collision Defeat** — hitting the rival's body segments triggers immediate Game Over, turning every session into a fast-paced spatial duel.
* Rival snake rendered in a contrasting accent colour to distinguish it from the player at a glance.

### 🎵 Arcade Synth Audio Engine
* `SoundManager.java` — programmatic MIDI sound engine using `javax.sound.midi`, zero external libraries.
* Square-wave synth voice (Lead 1) plays distinct notes for: food eat blip, Golden Apple eat, Shield Orb pop, and Game Over crash buzz.
* **Chiptune Synth Audio Effects** toggle checkbox in the sidebar mutes/unmutes all audio instantly.

### 🌟 Power-Up Target Variations
* **Golden Apple** (10 % spawn chance) — rare food that grows the snake by 2 extra body nodes and awards +3 points.
* **Shield Orb** (5 % spawn chance) — gives the snake a glowing blue shield bubble. Absorbs the next single obstacle or border collision, shattering the obstacle block and playing a break sound.

### 🎨 Visual Theme Customizer
* Sidebar combo-box allows real-time stylesheet swaps without restarting the game:
  * **Cyberpunk Neon** — glowing cyan and green on pure black (default).
  * **Vaporwave Pink** — pastel pink and neon cyan on dark violet.
  * **Matrix Green** — monochrome retro-terminal green palette.
* All theme colours are applied to the grid, snake body, food, borders, and HUD accents simultaneously.

### 🗺️ Map Creator & Obstacle Painter
* **Map Obstacle Editor Mode** checkbox pauses the game and enables mouse painting on the grid.
* Click or drag on empty cells to draw wall segments; click on existing walls to erase them.
* Supports building fully custom level layouts without leaving the application.

### 🛡️ UI Controls Refinements
* **Dynamic Pause/Resume button** — button label changes to **Resume** the instant the game is paused (click or SPACE/P shortcut), and switches back to **Pause** on resume. Timer stops completely during pause — no drift, no duplicate tick listeners.
* **Auto-pause safety** — opening the Leaderboard, Help, or Credits dialog automatically pauses an active game so the snake cannot crash in the background.
* **Exit button** — full-width dedicated Exit button below the action panel. Stops the game loop timer, closes database resources, and disposes the JVM window cleanly.

## 🏛️ Architecture Progress

**Battle-state management added to `GameController`.** The rival snake lives in a parallel data structure inside `GameController.java`, driven by a second AI steering call on every timer tick. Collision detection between the two snakes runs in O(n) per tick where n is the rival's body length.

**`SoundManager` as a stateless utility.** All audio calls are fire-and-forget static-style invocations; `SoundManager` manages its own internal MIDI channel state without requiring the caller to track resources.

Known limitations at this release:

* No automated test suite (see [tests/README.md](../../tests/README.md) for guidance).
* SQLite score log is a flat table — no per-session game-event timeline.
* Rival snake AI does not account for the player's projected path (greedy food chasing only).

# Smart Snake — v3.0.0

> *Codename: Synthserpent*

---

## 📖 Overview

**Synthserpent** is the architectural modernization release. The monolithic single-panel codebase from Cyberserpent is refactored into a strict Model-View-Controller separation, the window expands to 1,100 × 600 px, and a full neon sidebar dashboard is introduced alongside the dark cyberpunk visual theme that defines Smart Snake's look.

---

## 🎯 Objectives

* Refactor the monolithic game panel into clean MVC classes.
* Introduce a persistent sidebar HUD with live statistics.
* Expose runtime controls for controller mode, speed, and AI parameters.
* Establish the dark neon visual identity of Smart Snake.

---

## ✨ Added

### 🏗️ MVC Architecture
* `GameModel.java` — owns all game state: snake deque, food, direction, score, flags.
* `GameView.java` — pure renderer; reads `GameModel`, draws to `Graphics2D`.
* `GameController.java` — owns the `javax.swing.Timer`, drives AI steering and collision logic.
* `Project.java` — main `JFrame`; composes the three MVC layers and manages the sidebar.

### 📊 Neon Sidebar Dashboard
* Persistent right-side panel (300 px wide) with live statistics:
  * Score, High Score, Total Steps, A\* Path Length, Path Efficiency Index, Agent State Vector.
* Replaces the old in-canvas text overlays.

### 🎛️ Runtime Control Widgets
* **Controller Mode** radio buttons — switch between Manual, A\* Pathfinder, and Q-Learning mid-session.
* **Speed Slider** — maps 1×–5× to 100–20 ms timer delay without restarting the game.
* **Epsilon Slider** — adjusts Q-agent exploration rate (0.0–1.0) live.
* **A\* Path Overlay Toggle** — renders the planned path on the grid in real time.

### 🎨 Visual Identity
* Dark base background (`#0F1116`), neon cyan accents (`#00E5FF`), neon green values (`#39FF14`).
* Segoe UI typography throughout sidebar and dialogs.
* Hover effects on buttons: accent fill with high-contrast dark text.

---

## 🔄 Changed

* Refactored project source from a monolithic `JPanel` class into the four MVC files above.
* Window width extended from 800 px to 1,100 px to accommodate the sidebar.

---

## ⚠️ Breaking Changes

* Source structure incompatible with v2.0.0; direct class references to the old panel class are removed.

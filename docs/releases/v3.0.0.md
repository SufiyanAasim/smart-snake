# 🚀 Smart Snake v3.0.0
## Codename: Synthserpent — Dashboard Interface

**Release Date**: July 16, 2026

---

### Highlights
- Strict MVC architectural model separation.
- Expanded window dimensions (1100x600px) with dashboard sidebar panel.
- Dynamic control controls for speeds, paths, and training variables.

### Added
- Separated MVC packages: Model state (`GameModel.java`), graphics view (`GameView.java`), controller ticks (`GameController.java`).
- Control dashboard sidebar container providing live status changes.
- UI toggles to change controller type: Manual, A* Pathfinding, Q-Learning.
- Epsilon parameters and simulation tick rate sliders.
- Real-time path projection overlay drawn on the grid.

### UI/UX
- Sleek dark theme with neon borders and harmonious HSL color styling.
- Responsive statistic dashboards showing steps, efficiency, and state values.

### Breaking Changes
- Refactored project source structure from monolithic JPanels to MVC classes.

# Smart Snake — v2.0.0

> *Codename: Cyberserpent*

---

## 📖 Overview

**Cyberserpent** transforms Smart Snake from a manual game into an intelligent simulator. This release introduces two autonomous AI agents — an A\* pathfinder and a Q-learning reinforcement learning agent — alongside a dynamic obstacle system that increases difficulty as the score grows.

Building on the stable game loop from **Retroserpent**, Cyberserpent adds the algorithmic core that defines Smart Snake's identity as an AI showcase.

---

## 🎯 Objectives

* Integrate A\* pathfinding with Manhattan distance heuristics for optimal food routing.
* Add a BFS safety fallback that keeps the snake alive when A\* has no reachable path.
* Implement and train a compact Q-learning reinforcement learning agent.
* Introduce dynamic barriers that spawn progressively as the player's score increases.

---

## ✨ Added

### 🧠 A\* Pathfinder Autoplay
* `Pathfinder.java` implements A\* graph search using a Manhattan distance heuristic.
* Recalculates the shortest path to food on every game tick.
* Smooth real-time steering: the agent follows the computed next step automatically.

### 🛡️ BFS Safety Fallback
* When A\* cannot find a path to food, a BFS safety engine takes over.
* Steers the snake toward its own tail or the largest open space to avoid trapping.
* Prevents premature game-over in complex obstacle configurations.

### 🤖 Q-Learning Reinforcement Agent
* `QLearningAgent.java` implements tabular Q-learning over a compact 128-state space.
* State encodes: 3 danger bits (straight, left, right) + 4 relative food direction bits.
* ε-greedy action selection (straight, turn-left, turn-right).
* Headless training runs 10,000 episodes in under 3 seconds on the background thread.
* Weights persisted to `q_table.txt` and reloaded on next launch.

### 🧱 Dynamic Obstacle System
* Barrier blocks spawn on the grid every 4 points scored.
* Obstacles avoid the snake body, food, and previously placed barriers.
* Collision with a barrier triggers Game Over (unless a Shield Orb is active).

### ⚡ Connection Analysis
* Food spawn logic checks that the new food cell is reachable before placing it.
* Prevents the snake from getting stuck with no valid path in heavily obstructed maps.

## ⚡ Performance

* A\* pathfinding over the 1,200-cell grid completes in under 1 ms per tick.
* Q-learning trains to convergence in under 3 seconds (headless, background thread).

# 🚀 Smart Snake v2.0.0
## Codename: Cyberserpent — Intelligent Autoplay

**Release Date**: July 16, 2026

---

### Highlights
- Autoplay agent using A* pathfinding and BFS safety routing.
- Headless training engine for Q-learning reinforcement learning.
- Dynamic game elements including speed scaling and obstacles.

### Added
- Pathfinder engine (`Pathfinder.java`) implementing A* search using Manhattan distance.
- BFS safety search steering the snake to its tail when A* path is unreachable.
- Headless Q-learning simulation agent (`QLearningAgent.java`) and file persistence (`q_table.txt`).
- Collision boundaries and barrier blocks spawning as the player's score increases.
- Connection analysis to prevent food from spawning inside obstacles or loops.

### Performance
- Fast simulation: trains 10,000 Q-learning steps in < 3 seconds.
- Sub-millisecond pathfinding calculation latency.

### Compatibility
- Requires JDK 21+ and local write permissions for Q-table saving.

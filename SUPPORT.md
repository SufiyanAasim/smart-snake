# Support Guidelines

Thank you for playing and developing with Smart Snake! Here is how to get support.

## Issue Tracking
If you encounter a bug or have a feature suggestion, please open a GitHub Issue using one of our templates:
- **Bug Report Template** for game crashes or logic glitches.
- **Feature Request Template** for new pathfinders or dashboard widgets.

## Questions & Discussions
For general questions, design feedback, or help setting up local JDK 21 paths, please open a **GitHub Discussion** under the Q&A category.

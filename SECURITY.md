# Security Policy

## Supported Versions

Only the latest major version is supported for security updates:

| Version | Supported          |
| ------- | ------------------ |
| 4.x     | :white_check_mark: |
| < 4.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it immediately:
1. Do NOT open a public issue.
2. Email reports to: security@sufiyanaasim.com.
3. We will respond within 48 hours to coordinate a security patch.
